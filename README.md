## 🚀 Features

- ⚡ Fully responsive — works on mobile, tablet, and desktop
- 🖥️ Interactive terminal — type real commands (`whoami`, `skills`, `projects`, `help`, `clear`)
- 🎞️ Scroll animations — sections fade in as you scroll
- 📱 Mobile hamburger navigation menu
- 🔦 Active nav link highlighting on scroll
- 🌑 Dark hacker theme with scanline overlay effect
- 📄 Resume / CV download button
- 🔒 Zero dependencies — pure HTML, CSS, and vanilla JavaScript

## 🔧 Terminal Commands

Type these in the interactive terminal on the site:

```bash
whoami       # Who I am
skills       # My technical skills
projects     # My projects list
contact      # How to reach me
education    # My education details
location     # Where I'm based
date         # Current date/time
help         # All available commands
clear        # Clear the terminal
```

## 🌐 Deployment

This portfolio is deployed on **Vercel** as a static site.

To deploy your own copy:

1. Clone or download this repository
2. Make sure the file is named `index.html`
3. Place `index.html` and `image (2).jpeg` in the same folder
4. Drag the folder to [app.netlify.com/drop](https://app.netlify.com/drop) **or** push to GitHub and connect to Vercel
